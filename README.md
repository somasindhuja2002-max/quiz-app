 # 🧠 Quiz App (UI Design Project)
A clean and modern **Quiz Application UI** built using **HTML5 and CSS3**.  
This project focuses on designing a visually appealing and user-friendly quiz interface without using JavaScript.

---
## 🌐 Live Demo

Experience the project here:  
👉 [Launch Quiz App](https://somasindhuja2002-max.github.io/quiz-app/)

---

## ✨ Overview

This project is a front-end UI implementation of a quiz system.  
It demonstrates strong understanding of layout design, spacing, alignment, and responsive styling using CSS.

---

## 🚀 Features

- 🎯 Structured quiz layout with questions and options
- 🎨 Clean and modern UI design
- 📱 Responsive design for different screen sizes
- 🧩 Well-organized sections for questions and answers
- 🖌️ Styled buttons and interactive-looking elements (CSS only)

---

## 🛠️ Technologies Used

- **HTML5** – For semantic structure and content
- **CSS3** – For styling, layout, and responsiveness

---

## 🎯 What I Learned

- Structuring web pages using semantic HTML
- Creating visually balanced layouts using CSS
- Working with spacing, fonts, and colors effectively
- Designing UI components like buttons and quiz cards
- Improving responsive design thinking
